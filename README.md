# Dev-coconut94 Portfolio

개인 포트폴리오와 프로젝트 허브를 위한 GitHub Pages 공개 저장소입니다.

## 목적

- 공개할 정적 사이트만 별도로 관리
- 프로젝트 소개, 다운로드 허브, 노트 공개
- private 작업 저장소와 public 배포 저장소를 분리

## 공개 경로

- 홈: `/`
- 프로젝트: `/projects/`
- 다운로드: `/downloads/`
- 노트: `/notes/`

## 운영 방식

원본 작업과 초안은 private 저장소에서 관리하고, 이 저장소에는 공개 가능한 정적 결과물만 반영합니다.
